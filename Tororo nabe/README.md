# 🍲Troro nabe🥔

## Ingredients

・昆布出汁
・鳥つくね
・野菜(キャベツ、水菜、豆腐)
・きのこ(しいたけ、えのき)
・お麩
・豚肉
・牛肉
・くずきり
・とろろ
・なつめ

## Step1
鍋に水を入れ、昆布でお出汁を取っておく。

## Step2
①に火をかけ、温まってきたら、鶏つくね、なつめを入れ沸騰させる。

## Step3
②に野菜ときのこを入れ、ひと煮だちする。

## Step4
軽く沸騰している③に、豚肉を入れ火を通す。

## Step5
④全体に、とろろをかけ流す。

## Step6
完成✨まずは、そのままとろろ鍋を楽しみます。
